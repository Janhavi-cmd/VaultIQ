### рҹҺҜ Decision Intelligence Engine
Every stock gets a **proprietary Decision Score (0вҖ“100)** calculated from:

| Signal | Weight |
|--------|--------|
| RSI Momentum | 25% |
| Moving Average Crossover | 20% |
| Volatility Index | 20% |
| 52-Week Position | 20% |
| Volume Anomaly | 15% |

Stocks are then classified as:
- рҹҹў **BUY** вҖ” Strong momentum, low risk
- рҹҹЎ **HOLD** вҖ” Stable, wait for clearer signal
- рҹ”ҙ **RISKY** вҖ” High volatility, proceed with caution

## рҹ§Ә Engineering Highlights

- **Offline-First Architecture** вҖ” Synthetic data engine generates realistic OHLCV series, app works 100% without backend
- **Async Concurrent Fetching** вҖ” Uses `asyncio.gather()` to fetch all 60 stocks simultaneously
- **TTL Caching** вҖ” 5-minute in-memory cache reduces yfinance API calls by ~90%
- **Z-Score Anomaly Detection** вҖ” Statistical outlier detection flags unusual price/volume behavior
- **Modular Backend** вҖ” Clean separation: routing вҶ’ data вҶ’ ML
