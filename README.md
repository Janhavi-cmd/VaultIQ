- Unusual moves (|z| > 2.0) вҶ’ **High** severity

- Direction: рҹҡҖ **Spike** (positive) or рҹ’Ҙ **Crash** (negative)
- Every event has exact date, return magnitude, and z-score

## рҹ§Ә Engineering Highlights

- **Offline-First** вҖ” Synthetic data engine generates realistic OHLCV series; app works 100% without backend
- **Async Concurrent Fetching** вҖ” `asyncio.gather()` fetches all stocks simultaneously at startup
- **TTL Caching** вҖ” 5-minute in-memory cache + SQLite persistence reduces API calls by ~90%
- **WebSocket Live Ticker** вҖ” `/ws/ticker` broadcasts price updates every 2 seconds to all connected clients
- **NLP Query Engine** вҖ” Plain English вҶ’ structured API call (e.g. *"compare TCS and INFY 90 days"*)
- **Modular Backend** вҖ” `main.py` вҶ’ `data_service.py` вҶ’ `ml_predictor.py` clean separation
- **Indian Market Support** вҖ” NSE `.NS` suffix, вӮ№ INR pricing, BSE-aware sector classification
